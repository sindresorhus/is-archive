# is-archive

> Check if a file path is an archive file


## Install

```
$ npm install is-archive
```


## Usage

```js
const isArchive = require('is-archive');

isArchive('source/unicorn.zip');
//=> true

isArchive('source/unicorn.txt');
//=> false
```
